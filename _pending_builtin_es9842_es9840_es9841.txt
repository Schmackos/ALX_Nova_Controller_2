== Notes for integrating agent ==

- All 3 devices are expansion mezzanine devices (HAL_I2C_BUS_EXP = Bus 2, GPIO28/29).
- Default I2C address 0x40 for all three (configurable via ADDR pins).
- Chip ID distinction (register 0xE1):
    ES9842PRO = 0x83
    ES9840    = 0x87
    ES9841    = 0x91
    (ES9843PRO = 0x8F, already registered)
- ES9841 has different volume encoding (8-bit, 0xFF=0dB) vs ES9842PRO/ES9840 (16-bit, 0x7FFF=0dB).
- ES9842PRO and ES9840 are architecturally identical (same register map); ES9840 has lower DNR.
- All three output 4ch TDM and use HalTdmDeinterleaver (same as ES9843PRO).
- Driver source files:
    src/hal/hal_es9842pro.h + hal_es9842pro.cpp
    src/hal/hal_es9840.h    + hal_es9840.cpp
    src/hal/hal_es9841.h    + hal_es9841.cpp
    src/drivers/es9842pro_regs.h
    src/drivers/es9840_regs.h
    src/drivers/es9841_regs.h
